🎬 YouTube Integration

Add a free YouTube Data API key to search for your albums on YouTube, link tracks, and build playlists. Playlist creation also requires a YouTube OAuth Client ID.

How to get a free YouTube API Key (2 minutes):
1. Go to console.cloud.google.com → Create a project
2. Go to APIs & Services → Library → search and enable "YouTube Data API v3"
3. Go to APIs & Services → Credentials → Create Credentials → API Key
4. Copy the key and paste it below.
Free quota: 10,000 units/day — plenty for personal use.
